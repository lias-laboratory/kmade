KMADe is under development, that's why :
  • The enumeration and interval types are out of order (used to define a specific abstract attribut type);
  • Undo/Redo are not yet implemented;
  • Task caracteristics "Multimedia" "Goal" "Display" "Duration" are out of order;
  • ...
New features compared to the K-MADe 0.1.6 version :
  • New pie menu for task creation and task edition
  • Change of the graphic display of a task
  • Users (potential actors) are now Persons and Organizations;
  • Hardwares (potential system actors) are implemented. There are machine and machine stocks;
  • The decimal and negative number can be used;
  • Array are a new type of group;
