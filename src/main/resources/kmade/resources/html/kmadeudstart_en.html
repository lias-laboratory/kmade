KMADe is under development, that's why :
  • The enumeration and interval types are out of order (used to define a specific abstract attribut type);
  • The predicat expression for the iteration characteristic is not taken into account during the simulation process, only the variant iteration expression;
  • Cut/Paste/Copy are not yet implemented;
  • Undo/Redo are not yet implemented;
  • ...
Bugs meet with K-MADe 0.1.7 version :
  • QuickTime codec initialization;
  • ...
New features compared to the K-MADe 0.1.6 version :
  • New basic statistics
  • New printing features
    • Zoom/Unzoom on the print preview;
    • Print all task caracteristics;
    • Capability to customize the printing of task caracteristics thanks to the Velocity language.
  • Capability to save and restore the states of the concrete objects
    • Restore the states of the concrete objects during the edition of postcondition expression;
    • Undo during the simulation process.
  • Complete internationalisation : K-MADe is now available in english.
