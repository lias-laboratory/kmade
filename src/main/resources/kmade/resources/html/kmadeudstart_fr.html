KMADe est en développement, c'est pourquoi :
  • Les types énumération et intervalle (type d'un attribut abstrait) ont été désactivés;
  • L'itération façon prédicat n'est pas pris en compte dans la simulation;
  • Copier/Coller/Couper non implémentées;
  • Undo/Redo non implémentées;
  • Retour arrière dans simulation non implémentée;
  • ...
Différents problèmes généralement rencontrés avec la version K-MADe 0.1.7 :
  • Initialisation du codec QuickTime;
  • ...
Nouvelles caractéristiques depuis la version 1.6 :
  • Nouvelles statistiques basiques
  • Nouvelles fonctionnalités d'impression
    • Agrandissement/Dégrossissement sur la vue de l'impression;
    • Impression des caractéristiques de tâches;
    • Possibilité de paramétrer l'affichage des caractéristiques à l'aide du langage Velocity.
  • Possibilité de sauvegarder et restaurer les états des objets concrets
    • Restaure les états des objets concrets pendant l'édition de la postcondition;
    • Retour arrière pour le processus de simulation.
  • Internationalisation de K-MADe : K-MADe supporte la langue de Shakespeare!!
